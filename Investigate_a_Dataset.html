Introduction¶

This data set is from TMDB ( The Movie Data Base ) contains data about more than 10,000 movies ( exactly 10866 movies ). it contains the full credits for both the cast and the crew in addition to popularity , budget, revenu and other factors.

The goal here is to find an answer to a few questions like :

  • what is the relationship between the popularity of a movie and its revenu ?
  • whether movies with higher budget are more popular ?

Data Description¶

id : id of the movie

imdb_id : IMDB id of the movie

popularity : popularity of the movie

budget : budget in Us dollars

revenue : revenue in Us dollars

original_title : title of the movie

cast : full cast

homepage : website homepage of the movie

director : the name of the director/directors

tagline : tagline of the movie

keywords : keywords to describe the movie

overview : overview of the movie

runtime : how many that movie run

genres : genres of the movie seperated by pipes ( | )

production_companies : name of production company/ companies

release_date : full date of release

vote_count : how many votes

vote_average : the average of the votes

release_year : the year of release

budget_adj : budget of the movie in terms of 2010 dollars

revenue_adj : revenu of the movie in terms of 2010 dollars

Conclusions¶

first we imported our data, then we made a data wrangling process from cleaning to removing unwanted properties/columns whether for null values or unwanted columns.

then we plotted some scatter plots to check the correlation between the wanted properties to find answers to our questions.

finally we can say that there is a positive correlation between each property of the following ( popularity vs revenue_adj & budget_adj vs popularity & budget_adj vs revenue_adj), also with the histogram plots of each property we notice that it is normally distributed individually. and that means that the more the budget of the movie the more its popularity and its incoming revenue
